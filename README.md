# Malala Yousafzai - Landing Page

## Sobre a personagem
Malala Yousafzai é uma ativista paquistanesa que luta pelo direito à educação, especialmente para meninas.

## Desenvolvimento da página
A página foi desenvolvida utilizando HTML para estrutura, CSS para estilização e JavaScript para interatividade.

## Uso do JavaScript
Foi utilizado um botão que exibe uma curiosidade sobre Malala ao ser clicado.

## Organização do projeto
Os arquivos estão organizados em:
- index.html (estrutura)
- style.css (estilo)
- script.js (interação)
- pasta imagens (imagens utilizadas)
